Exercicio para a Jack

Lista de requisitos que vc precisa cumprir

url do back endpoint de pesquisa https://teste-da-jack.herokuapp.com/search?name=a&quantity=20

  • precisa estar em React!!
  • ter uma tela com um campo com o nome "pesquise seu pokemon
  • quando o user digitar no campo, a aplicação deve pesquisar na api usando o valor que está no campo
  • precisa mostar nome do pokemon e tipo
  • só fazer a pesquisa depois que o usuário digitou no minimo 2 caracteres
  • esperar 0.5 s depois do usuário para de digitar, só depois desse tempo que a aplicação deve fazer a request para a api(Dica, pesquise "debouncing function")
  • os resultados da pesquisa devem aparecer abaixo do campo
  • não precisa ficar bonito, pode deixar sem estilo mesmo
